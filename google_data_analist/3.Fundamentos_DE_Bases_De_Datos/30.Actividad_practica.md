# Actividad práctica: Elija la herramienta adecuada para el trabajo

## Visión general de la actividad

Como profesional del análisis de datos, parte de su trabajo consistirá en comprender qué herramientas son las adecuadas para el trabajo. Llegados a este punto, ya se ha familiarizado con las hojas de cálculo y SQL como herramientas potenciales para el análisis de datos. Esta actividad es una oportunidad para aplicar sus nuevos conocimientos sobre las herramientas de análisis de datos a escenarios de trabajo. Después de leer cada escenario, se le pedirá que elija si utilizaría una hoja de cálculo o SQL para trabajar con los datos y que explique por qué ha seleccionado esa herramienta.
